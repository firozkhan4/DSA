### **Easy Array Questions:**

1. **Find the Maximum and Minimum element in an array.**
2. **Reverse an Array.**
3. **Find the "Kth" largest/smallest element in an array.**
4. **Find the sum of all elements in an array.**
5. **Find the occurrence of a given element in an array.**
6. **Remove duplicates from a sorted array.**
7. **Move all zeroes to the end of an array.**
8. **Rotate an array by "K" positions.**
9. **Find the missing number in an array of size N, containing numbers from 1 to N+1.**
10. **Merge two sorted arrays into a single sorted array.**

### **Hard Array Questions:**

1. **Find the maximum product subarray.**
2. **Find the maximum sum of two non-overlapping subarrays.**

4. **Merge Intervals problem (merge overlapping intervals).**
5. **Count the number of subarrays with a given XOR value.**
6. **Find the maximum length of a subarray with the sum equal to zero.**
7. **Find the minimum number of swaps to sort the array.**
8. **Find the longest subarray with at most K distinct elements.**
9. **Median of two sorted arrays of different sizes.**
10. **Find the number of inversions in an array (using Merge Sort).**
